# 📊 Insurance Sales & Revenue Analysis

## 🧩 Problem Statement

Analyze insurance sales data to track revenue, customer acquisition, and sales performance.

---

## 🎯 Objectives

* Monitor premium and revenue trends
* Analyze agent performance
* Track sales funnel stages

---

## 🛠 Tools Used

* MySQL
* Power BI
* Excel

---

## 🔄 Project Workflow

1. Extracted and cleaned data using MySQL
2. Performed analysis on sales and revenue
3. Built interactive dashboard in Power BI

---

## 📊 Key Insights

* Majority revenue comes from specific agents
* Sales pipeline shows bottlenecks in conversion stages
* Increase in meetings led to higher sales activity
