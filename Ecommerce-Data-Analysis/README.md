# 🛒 E-commerce Sales Data Analysis (Python + MySQL)

## 🧩 Problem Statement

Analyze e-commerce sales data to identify trends, top-performing products, and customer behavior using MySQL and Python.

---

## 🎯 Objectives

* Extract and analyze sales data from MySQL database
* Understand sales trends over time
* Identify top products and categories
* Analyze customer purchasing behavior

---

## 🛠 Tools & Technologies

* MySQL (Data Storage & Querying)
* Python (Pandas, Numpy, Matplotlib)
* Jupyter Notebook

---

## 🔄 Project Workflow

1. Connected MySQL database with Python
2. Extracted data using SQL queries
3. Performed data cleaning and preprocessing in Python
4. Conducted Exploratory Data Analysis (EDA)
5. Visualized key trends and insights

---

## 🔍 Analysis Performed

* Data cleaning and transformation
* Sales trend analysis
* Category-wise performance
* Customer behavior analysis

---

## 📊 Key Insights

* Certain categories contribute majority of revenue
* Sales show seasonal patterns
* A small group of customers drives large revenue share
* Some products have high sales but lower profitability

---

## 📂 Files Included

* Notebook: `Notebook/ecommerce-analysis.ipynb`
* HTML: `HTML/ecommerce-analysis.html`
* Dataset: `Dataset/ecommerce-data.csv`
