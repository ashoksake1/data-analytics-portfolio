# 🏭 Manufacturing Performance & Quality Analysis

## 🧩 Problem Statement

Analyze production data to monitor efficiency, output, and product quality.

---

## 🎯 Objectives

* Track production quantity
* Analyze rejection rates
* Identify machine performance issues

---

## 🛠 Tools Used

* MySQL
* Power BI
* Excel

---

## 🔄 Project Workflow

1. Queried and prepared production data using MySQL
2. Analyzed rejection and output trends
3. Built dashboard in Power BI and Excel

---

## 📊 Key Insights

* Certain machines show higher rejection rates
* Specific product categories dominate production
* Rejection trends highlight quality control issues
